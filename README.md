# 🏛️ Divine Judgment: Mankind's Last Stand

An action-packed, 1v1 mythical combat game built for Android using **Unity**. Players take on the role of humanity's chosen champion to battle supreme deities like Zeus and Thor in a tournament that determines the fate of mankind.

---

## 🌌 The Lore
In a world where the balance between gods and humans has broken, the divine council decides humanity is corrupt and votes for extinction. However, a final proposal is made: humanity is given one last chance to prove its worth through a tournament. The strongest gods face legendary human warriors in intense one-on-one combat. Survival depends on every victory.

---

## 🛠️ Key Features
* **Dynamic 1v1 Combat Engine:** Responsive, combo-based attack and defense mechanics tailored for mobile.
* **Adaptive Boss AI:** Scaling difficulty algorithms that adapt to player behavior, making god battles uniquely challenging.
* **Progressive Skill System:** Unlock and upgrade skills as the game advances to match the increasing strength of the gods.
* **Optimized for Android:** Fully optimized asset pipeline and code architecture ensuring stable frame rates (60 FPS) and smooth gameplay on mobile devices.

---

## 🚀 Tech Stack & Tools
* **Game Engine:** Unity 6000.3.15f1 
* **Language:** C#
* **Version Control:** Git

---

## 📋 Architecture & Implementation Details
* **State Machine Pattern:** Used for managing complex AI behaviors (Idle, Patrolling, Attacking, Stunned, Defeated) for both the player and the enemy deities.
* **Event-Driven UI:** Decoupled gameplay systems from the UI components using C# Actions/Events for better performance and maintainability.
* **Mobile Input System:** Utilized Unity's New Input System for seamless touch screen joystick and button configurations.
